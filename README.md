# Гайд по работе с сервером

## Подключение к серверу

1. Открой **PowerShell** (Win+R, напиши `powershell`, Enter)
2. Введи команду:
```
ssh root@202.181.188.42
```
3. При первом подключении спросит `Are you sure?` — напиши `yes` и нажми Enter
4. Введи пароль (он не отображается при вводе — это нормально, просто печатай и жми Enter)

Ты на сервере, когда видишь строку вида:
```
root@server:~#
```

## Что такое screen

Screen — это "виртуальное окно". Сервер и лаунчер работают внутри screen-сессий. Если ты закроешь консоль — screen продолжит работать, и сервер не выключится.

### Основные команды screen

| Команда | Что делает |
|---------|-----------|
| `screen -ls` | Показать все запущенные screen-сессии |
| `screen -S имя` | Создать новый screen с именем |
| `screen -r имя` | Подключиться к существующему screen |
| `Ctrl+A, потом D` | Выйти из screen, не останавливая его |

## Запуск игрового сервера

### Если screen уже есть

1. Посмотри список screen-сессий:
```
screen -ls
```
2. Подключись к screen сервера:
```
screen -r server
```
3. Если сервер не запущен, запусти:
```
cd /home/server
sh start.sh
```

### Если screen нет (первый запуск)

1. Перейди в папку сервера:
```
cd /home/server
```
2. Создай screen с именем `server`:
```
screen -S server
```
3. Запусти сервер:
```
sh start.sh
```
4. Выйди из screen, не выключая сервер:
```
Нажми Ctrl+A, потом нажми D
```

## Запуск лаунчсервера

### Если screen уже есть

1. Подключись к screen лаунчера:
```
screen -r launcher
```

### Если screen нет (первый запуск)

1. Перейди в папку лаунчера:

cd /home/launcher

2. Создай screen с именем `launcher`:
```
screen -S launcher
```
3. Запусти лаунчсервер:
```
sh start.sh
```
4. Выйди из screen:
```
Нажми Ctrl+A, потом нажми D
```

## Частые ситуации

### Хочу посмотреть консоль сервера
```
screen -r server
```
Чтобы выйти обратно: `Ctrl+A`, потом `D`

### Хочу перезапустить сервер
```
screen -r server
```
В консоли сервера напиши `stop` — сервер выключится. Потом:
```
sh start.sh
```

### Хочу выключить сервер
```
screen -r server
```
Напиши `stop` в консоли сервера.

### Ошибка "There is no screen to be resumed"
Значит screen не создан. Создай его:
```
screen -S server
```

### Ошибка "There are several suitable screens"
Выполни `screen -ls`, посмотри точное имя и подключись по полному имени:
```
screen -r 12345.server
```

---

## Как выйти с сервера
Просто напиши:
```
exit
```
Screen-сессии продолжат работать.

## Работа с файлами через FileZilla (SFTP)

FileZilla — программа для удобной работы с файлами на сервере. Можно перетаскивать файлы мышкой, как в обычном проводнике.

### Установка

1. Скачай FileZilla Client с официального сайта: https://filezilla-project.org
2. Установи, все настройки по умолчанию

### Подключение к серверу

1. Открой FileZilla
2. В верхней панели заполни поля:

| Поле | Что писать |
|------|-----------|
| **Хост** | `sftp://202.181.188.42` |
| **Имя пользователя** | `root` |
| **Пароль** | твой пароль от сервера |
| **Порт** | `22` |

3. Нажми **"Быстрое соединение"**
4. Если выскочит окно "Неизвестный ключ хоста" — поставь галочку "Всегда доверять" и нажми **ОК**

### Интерфейс FileZilla

После подключения увидишь 4 панели:

| Панель | Что показывает |
|--------|---------------|
| Левая верхняя | Папки на **твоём компьютере** |
| Левая нижняя | Файлы на **твоём компьютере** |
| Правая верхняя | Папки на **сервере** |
| Правая нижняя | Файлы на **сервере** |

### Как перейти в нужную папку на сервере

В правой верхней панели есть строка с путём. Впиши нужный путь и нажми Enter:

| Куда | Путь |
|------|------|
| Игровой сервер | `/home/server` |
| Лаунчсервер | `/home/launcher` |
| Сайт (WebMCR) | `/var/www/webmcr` |
| Домашняя папка | `/root` |

### Как загрузить файл на сервер

1. Слева найди файл на своём компьютере
2. Справа открой нужную папку на сервере
3. Перетащи файл слева направо

### Как скачать файл с сервера

1. Справа найди файл на сервере
2. Слева открой папку на своём компьютере, куда хочешь сохранить
3. Перетащи файл справа налево

### Как редактировать файл на сервере

1. Найди файл на сервере (правая панель)
2. Нажми правой кнопкой мыши -> **"Просмотр/Правка"**
3. Файл откроется в блокноте
4. Внеси изменения, сохрани (Ctrl+S)
5. FileZilla спросит "Загрузить файл обратно на сервер?" — нажми **Да**
